# Comparações de Personagens (Vingadores vs Thanos)

Este projeto cria objetos em JavaScript para os Vingadores principais e o vilão Thanos, agrupa-os em um array e realiza comparações ponto a ponto entre os atributos numéricos de velocidade, força e resistência.

## Como rodar

Pré-requisito: Node.js instalado.

No Linux/bash, execute:

```bash
node R1/src/avengers.js
```

A saída mostrará, para cada par de personagens:
- As descrições completas de ambos (via método `descricao()`)
- Quem vence em velocidade, força e resistência (ou empate)

## Estrutura
- `R1/src/avengers.js`: Implementação dos personagens, agrupamento e lógica de comparação.
